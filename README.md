# tiny

> I started this project to try the new shadcn CLI and mess with trpc and prisma. Well, now it's a URL shortner project.

### Built with

- Next.js
- TailwindCSS
- PostgreSQL
- shadcn/ui
- prisma
- supabase
- trpc
- zod

### Setup

- Update the `DATABASE_URL` env variable in `.env` with your local PostgreSQL database.

```zsh
bun install
bun run db:migrate
bun run dev
```
